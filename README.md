# Invisible Women - Interactive React Presentation

An engaging, interactive React presentation based on my notes from "Invisible Women" by Caroline Criado Perez.

## Features

✨ **Interactive Slideshow**

- Navigate through 13 chapters with 100+ individual notes
- Smooth animations and transitions
- Color-coded chapters for easy identification

🎯 **Navigation Options**

- Click Previous/Next buttons
- Use keyboard arrow keys (← →)
- Jump to any chapter from the sidebar
- Progress bar shows your position in the presentation

### Installation

```bash
# Install dependencies
npm install

# Start the development server
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## How to Use

1. **Navigate Slides**: Use the Previous/Next buttons or arrow keys
2. **Jump to Chapters**: Click any chapter in the sidebar to go directly there
3. **Track Progress**: Watch the progress bar at the bottom
4. **View Details**: Each slide shows your current position (Chapter X, Note Y)

## Structure

The presentation is organized into the following chapters:

1. **The Problem**
2. **Unpaid Labor**
3. **Public Spaces**
4. **Work & Economics**
5. **Meritocracy Myth**
6. **Medical Neglect**
7. **The Henry Higgins Effect**
8. **The Sea of Dudes**
9. **Yentl Syndrome**
10. **A Costless Resource**
11. **From Purse to Wallet**
12. **Who Will Rebuild?**
13. **Afterword**

## Keyboard Shortcuts

- **→** Next slide
- **←** Previous slide
- **Click** Navigate and select chapters
